Keep in mind...

Go right if you can; go left if you must

  • Ref The expression &e has type t* if e has type t
  • Deref The expression *e has type t if e has type t*
  • Array The expression e1[e2] has type t if e1 has type t* or type t[] and e2 has type int
  • Struct The expression e.ai has type ti if e has type struct{t1 a1;...; tm am;}
  • App The expression e(e1,..., em) has type t if the return type of e is t and e1,..., em are subtypes of the types of the parameters in the function definition in the environment
  • Op ◻ The expression e1 ◻ e1 has type t if e1 and e2 have type t
  • Op = The expression e1 = e2 has type t1 if e1 has type t1 and a2 has type t2 and t2 is a subtype of t1
  • Explicit Cast The expression (t1)e has type t1 if e has type t2 and t2 is a subtype of t1
